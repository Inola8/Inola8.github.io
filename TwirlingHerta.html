About the game

Game

This is my take on a classic Kurukurukururin, the mechanics are the same: spin around and go to the next level. But I added some pickups and made my own art (inspired by another game character - Herta who likes to twirl). This was my first project!

    Unlockable levels

    In this code snippet I set all the levels to locked, except for the first one, I then check via PlayerPrefs if the next levels have been unlocked (which get updated in each level). Then I'll make the next level unlockable by making the button interactable.

    The ButtonsToArray function was not necessary but I wanted to try it out, so here I add all the buttons to an array, I loop through them and put them on a set position. This could be useful if I wanted to add more levels, so it would automatically update the position.

    Player health

    As for the player health I made heart Icons for each state, so I checked per health state which Icon to enable and disable, and added some logic upon dying.
