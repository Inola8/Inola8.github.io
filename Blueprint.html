About this project

Research

For this project I did research on how to make my own Blueprint Class using C++.
So this is a step by step project, I wrote a page about the process more thoroughly here. (However, it is in Dutch).
Step 1 was doing research, so let's jump straight into step 2 below.

My steps

  1. Research C++
  2. Make a Class that returns something
  3. Make a Class with a timer
  4. Make a Class that spawns an object after a certain time
  5. Make a Class that starts a level sequence after a certain time

Adding a timer

To make use of the OnTimer event from Unreal Engine, I had to include TimerManager. Which will make it possible to call and cancel a timer.
Here I did add a parameter and set it equal to the timer I will actually call in the TimerManager

The class in Blueprints

In Blueprints the user can add a float and give it a value to determine how long the delay take.
