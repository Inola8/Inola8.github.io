About the game

Game

This was a group project with 3 devs and one client/ artist from the film academy in Amsterdam. The request of this artist was to make an experience about worrying too much. So this takes place in a hospital with many doors that lead you to patients in need. The doors have timers, which get more and more to a point you cannot help them in time. The patients will die if not taken care of before the timer reaches it's end.
At the end of the day the player will have to realize it was all in their head, and it was just a regular day at work.
We had 8 weeks for this project, again this project must include a meta human, I worked on this project while also working on Overthinking To Socialize.

My efforts

    • Code

    Game timer

    I made a timer that determines the duration of the game, so this is just a timer that counts up until it reaches the max time, then it will spawn the exit door, where the player may end their shift. I do this by removing one of the doors, and place the exit door on the same position.
    At some point in the game I wanted to change the dust particle to a blood dust, for a small but dramatic effect.

    Lights

    Everytime a patient dies the lights in the main room will get darker to raise the player's anxiety. I get all the lights and put them in an array, then grab their light component and change the intensity.

    Blood on hands

    I added all the materials to the hands and change the index of the material every time a patient dies.

    Door pt. 1

    I based a lot of the logic for the doors on whether a patient needs help or not. Basically a lot of visual feedforward and feedback.

    Door pt. 2

    Here I call the Dim Lights function and Blood On Hands function I showed earlier.
    Since the player uses an interface to check the door, I cannot simply use a bool check. So I figured I just put an invisible wall infront of the doors when the patient dies, which will block the ray to the door.
