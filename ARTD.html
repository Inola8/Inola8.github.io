About the game

Game

This is a rather classic tower defense game, however it is entirely in VR. No controllers needed, you can touch and grab everything! We worked on this for 4 weeks with a total of 3 devs, later we got a couple of artists for a very short time to provide us with some assets. My focus was on making the game and gameloop, the others worked on the AR part.

    GameManager

    In the GameManager I took care of the start and end of the game, and the start of each round. For the start of the game, I simply enable the game itself after the start button has been pressed. As for the end, you can win or lose. When you win, firework will play, but when you lose an explosion will blow up the whole game board.
    In StartNewRound I keep the WaveManager up to date.

    WaveManager

    Everytime a new wave is activated the amount will increase and the enemies will be able to spawn. The enemies spawn at an interval, and I randomly pick an enemy that I split in two groups (so group one is easier, group two is harder).
    I keep track of the enemies in a List, when they are spawned they are added, when they die they will be removed. When an enemy is removed I will check if the List is empty, so I can activate the next wave.

    Shop

    The player can buy powerups in this shop, upon pressing a button I check if the player has enough coins to get the powerup. If this is true, then i retract the cost and activate the powerup.
    I connect the powerup though an int which is the parameter, so all the powerups have their own int, which is compared when this function is called.
